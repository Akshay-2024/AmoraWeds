# ✨ Amora Weds

💍 A modern luxury wedding invitation website built using **Next.js**, **React**, and immersive UI animations.  
Designed for premium digital wedding experiences with elegant visuals, smooth interactions, and responsive layouts.

---

## 🌐 Live Demo

🔗 https://amora-weds.vercel.app/

---

## 🚀 Features

✔️ Premium wedding invitation UI  
✔️ Fully responsive design  
✔️ Smooth animations & transitions  
✔️ Interactive modern layouts  
✔️ RSVP-ready structure  
✔️ Fast performance with Next.js  
✔️ Mobile-friendly experience  
✔️ Easily customizable themes
